= Appendix

.Priority Definitions
|===
| Priority | Definition

| **P0**
| Must have — MVP blocking; core functionality required for launch
| **P1**
| Should have — Important for v1.0 or early post-launch; significant user value
| **P2**
| Nice to have — Enhancement; can be deferred to post-launch roadmap
|===
