If you want to exclude AI providers using your project for training then you could do the following...

Include empty file `.github/copilot-ignore`

Set `.gitattributes`

[source,gitignore]
----
*.java ai-training=false

*.adoc ai-training=false
----

IMPORTANT: Given their track record there is obviously no guarantee that these companies will follow those indications. And keep in mind: this is a double edged sword. Projects like https://javaparser.org/[Java Parser] might also not be able to use your project to e.g. test.
